# Paper Age GUI

Rust base and WASM compatible GUI for [paper-age](https://github.com/matiaskorhonen/paper-age).

## Installation

### From source

You need to have cargo installed.

```bash
git clone "https://github.com/renesat/paper-age-gui"
cd paper-age-gui

# Release build
cargo build --release

# Just run
cargo run
```
